House to Home

House to Home” the goal of this website is to eliminate the use of realtors. Buying property is expensive as is, so why include a realtor? Realtor’s can cost thousands of dollars, so we decided to build an immersive platform that allows any buyer to be their own realtor. Buyers will be able to interact with the sellers directly, there will also be a system manager that keeps track of all listings and user finances.
